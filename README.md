# [EditorMD](https://dt27.org/php/editormd-for-typecho/)

**Markdown 编辑器 [Editor.md](https://pandao.github.io/editor.md/) for Typecho**

## Features
- 支持实时预览、代码/表格插入、代码折叠等功能；
- 支持 [Emoji 表情](http://www.emoji-cheat-sheet.com/)；
